# 🎯 竞技场射手 (Arena Shooter 3D)

纯浏览器运行的 3D 第一人称射击游戏，无需安装任何东西。

## 🎮 快速开始

1. 下载本仓库，双击 `index.html`（或浏览器打开）
2. 点击 **单人游戏** 开始

> 或在线玩：https://qiguaidhanhan.github.io/arena-shooter/

## 🕹️ 操作

| 按键 | 功能 |
|------|------|
| WASD | 移动 |
| 鼠标 | 瞄准 |
| 左键 | 射击 / 刺击 |
| R | 换弹（12发弹匣，无限备弹） |
| 空格 | 跳跃 |
| 滚轮 | 切换手枪 / 匕首 |
| C | 烟雾弹 |
| Q | 上升气流 |
| E | 冲刺 |
| X | 飞刀风暴（大招，6杀充能） |

## 🤖 敌人兵种

| 兵种 | 特点 |
|------|------|
| 小兵 | 基础均衡 |
| 快跑者 | 高速低血 |
| 重装兵 | 厚血重火力 |
| 狙击手 | 超远射程 |
| 医疗兵 | 治疗友军 |
| 重盾兵 | 前置大盾 |
| 自爆兵 | 近身爆炸 AOE |
| 幽灵 | 半透明潜行 |
| 炮塔 | 固定双枪管 |

## 🌐 双人联机

主机：
```bash
python server.py    # 或 node server.js
# 浏览器打开 index.html → 创建房间
```

客机：
```
浏览器打开 index.html → 输入主机IP:8765 → 加入房间
```

纯 PvP 对战，无 AI 干扰，互相伤害。

## 🛠️ 技术栈

- Three.js (3D 渲染)
- Web Audio API (音效)
- WebSocket (联机)
- 纯 HTML/CSS/JS，零依赖
