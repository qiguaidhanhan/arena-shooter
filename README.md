# 💀 竞技场射手 (Arena Shooter 3D)

纯浏览器运行的 3D 第一人称射击游戏，无需安装任何东西。

## 🎮 快速开始

1. 下载本仓库，双击 `index.html`（或浏览器打开）
2. 点击 **单人游戏** 开始

> 或在线玩：https://qiguaidhanhan.github.io/arena-shooter/

## 🕹️ 操作

| 按键 | 功能 |
|------|------|
| WASD | 移动 |
| 鼠标 | 瞄准 |
| 左键 | 射击 / 刺击 |
| R | 换弹（12发弹匣，无限备弹） |
| 空格 | 跳跃 |
| 滚轮 | 切换手枪 / 匕首 |
| C | 烟雾弹 |
| Q | 上升气流 |
| E | 冲刺 |
| X | 飞刀风暴（大招，6杀充能） |

## 📋 游戏特性

- 爆头 3 倍伤害判定
- 子弹散布基于移动速度（越跑越不准）
- 击杀敌人充能大招
- 50 种敌人 + 动态梯队波次系统
- 复杂 3D 地图（建筑、走廊、高台）
- 环境粒子 + 弹壳弹出 + 弹孔 + 暗角画质效果

## 🌐 双人联机

主机：
```bash
python server.py    # 或 node server.js
# 浏览器打开 index.html → 创建房间
```

客机：
```
浏览器打开 index.html → 输入主机IP:8765 → 加入房间
```

纯 PvP 对战，无 AI 干扰，互相伤害。

## 🛠️ 技术栈

- Three.js (3D 渲染)
- Web Audio API (音效)
- WebSocket (联机)
- 纯 HTML/CSS/JS，零依赖
